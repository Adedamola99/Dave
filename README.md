# Dave Akpan — Premium Fashion Model Portfolio

An award-winning, luxury-tier model portfolio website designed to feel like a high-end fashion campaign. Built from the ground up with vanilla HTML, CSS, and JS to achieve cinematic Awwwards-level interactions.

## 🌟 Key Features

- **Cinematic Loader:** A sophisticated loading screen that builds anticipation before wiping away to reveal the hero section.
- **Custom Cursor:** Dual-layer magnetic cursor with intelligent state changes (hover, drag).
- **GSAP-Quality Scroll Interactions:**
  - `IntersectionObserver`-powered reveal engine.
  - Text line masking animations (editorial style wipe-ups).
  - Advanced CSS spring physics for butter-smooth transitions.
- **Immersive Hero:** Full-screen slideshow with scroll-driven parallax imagery.
- **Masonry Portfolio:** True interlocking CSS masonry grid, complete with an interactive full-screen lightbox.
- **Draggable Editorial Feed:** Momentum-based horizontal scrolling for press features (supports both mouse drag and touch).
- **Premium Typography:** High-contrast pairing of *Cormorant Garamond* (serif) and *Montserrat* (sans).

## 🛠️ Tech Stack

This project was intentionally built without heavy frameworks to maximize performance and demonstrate core frontend mastery:

- **HTML5:** Semantic, accessible structure with ARIA labels.
- **CSS3:** Custom variables, grid/flexbox layouts, complex keyframes, and `cubic-bezier` easings.
- **Vanilla JavaScript (ES6+):** Complete custom animation engine, parallax handlers, momentum scrolling, and custom cursor logic.

## 🚀 Setup & Usage

Since this is a static site with no build tools or dependencies, simply open the project in any browser:

1. Clone the repository.
2. Open `index.html` in your web browser of choice.

*(For the best development experience, consider using an extension like VSCode Live Server to preview changes locally).*

## 🎨 Design Philosophy

Inspired by luxury fashion houses (Saint Laurent, Prada, Jacquemus) and elite creative studios (Obys, Locomotive), this site prioritizes:

1. **Elegance over noise:** Dark premium aesthetic (`#07070a` backgrounds) with warm ivory and muted gold accents.
2. **Intentional Motion:** Every scroll and hover feels deliberate. No cheap bouncy animations; only smooth, easing-driven motion.
3. **Editorial Whitespace:** Generous padding and dramatic scale changes to let the photography breathe.

## 📱 Responsiveness

Fully responsive across all devices:
- Gracefully degrades the complex grid layouts to simpler stacks on mobile.
- Replaces the custom hover cursor with native touch interactions on coarse-pointer devices.
- Includes a smooth, animated mobile-navigation drawer.
